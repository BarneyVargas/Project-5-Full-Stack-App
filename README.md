# Full-Stack Task Tracker  
**React + Express + PostgreSQL (Supabase)**

This project is a full-stack CRUD application that allows users to create, read, update, and delete tasks. It uses a React frontend built with Vite, an Express API backend, and a PostgreSQL database hosted on Supabase.

---

## Features

- Create, read, update, and delete tasks
- React frontend using hooks (`useState`, `useEffect`)
- Component-based UI with props
- Express REST API
- PostgreSQL database (Supabase)
- Environment variables secured with `.env`
- Deployed to cloud platforms

---

## Tech Stack

### Frontend
- React
- Vite
- JavaScript

### Backend
- Node.js
- Express
- PostgreSQL
- `pg` library
- Supabase (database hosting)
